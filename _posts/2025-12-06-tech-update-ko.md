# 🛠️ 2025-12-06 기술 업데이트 요약

## 🔹 Kubernetes - Kubernetes v1.35 Sneak Peek
Kubernetes v1.35의 출시를 앞두고, 프로젝트는 지속적으로 발전하고 있습니다. 이번 블로그 글에서는 v1.35 릴리스에서 예정된 주요 변경 사항을 소개합니다. 주요 변경 사항으로는 cgroup v1 지원이 제거되고, kube-proxy의 ipvs 모드가 폐기될 예정입니다. 또한, containerd v1.x 지원이 종료되므로, 사용자들은 containerd 2.0 이상으로 업그레이드해야 합니다. 주요 기능 개선 사항으로는 노드 선언 기능, Pod 리소스의 인플레이스 업데이트, Pod 인증서 기능 등이 포함됩니다. 이 릴리스는 2025년 12월 17일로 예정되어 있으며, 최신 개발 사항을 유지하기 위해 변경 사항을 주의 깊게 살펴볼 필요가 있습니다.
👉 [자세히 보기](https://kubernetes.io/blog/2025/11/26/kubernetes-v1-35-sneak-peek/)

## 🔹 Spring Boot - Spring AI 1.1.1 Available Now
Spring AI 1.1.1 버전이 출시되었습니다. 이번 릴리스에는 모델 제공자 기능 확장을 위한 13개의 새로운 기능, 안정성을 향상시키기 위한 16개의 버그 수정, 3개의 문서 개선, 5개의 의존성 업그레이드(2개의 보안 패치 포함)가 포함되어 있습니다. 새로운 기능으로는 OpenAI Java SDK와의 통합, ChatClient의 구조화된 출력 지원, Claude Skills API와의 파일 기반 상호작용 지원 등이 있습니다. 또한 Google Gemini 통합 개선, MCP 안정성 수정, 여러 버그 및 보안 문제 해결이 이루어졌습니다. Spring AI 2.0.0-M1은 다음 주에 출시될 예정입니다. 이번 릴리스는 많은 커뮤니티 기여자들의 도움으로 가능하게 되었습니다.
👉 [자세히 보기](https://spring.io/blog/2025/12/05/spring-ai-1-1-1-available-now)

## 🔹 Docker - Docker, JetBrains, and Zed: Building a Common Language for Agents and IDEs
이 기술 블로그 글은 에이전트가 코드 작성 및 리팩토링을 할 수 있게 되면서, 개발자가 작업하는 환경인 에디터 내에서 자연스럽게 작동해야 한다는 점을 강조합니다. 이를 위해 JetBrains와 Zed는 ACP(Agent Client Protocol)를 공동 개발하고 있습니다. ACP는 에이전트와 에디터가 공유할 수 있는 언어를 제공하여, 어떤 에이전트든 맥락을 읽고, 행동을 취하고, 지능적으로 반응할 수 있게 합니다. 이는 맞춤형 연결 없이도 가능하도록 설계되었습니다.
👉 [자세히 보기](https://www.docker.com/blog/docker-jetbrains-and-zed-building-a-common-language-for-agents-and-ides/)
