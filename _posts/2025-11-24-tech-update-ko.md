## 🔹 Kubernetes - Ingress NGINX Retirement: What You Need to Know
Kubernetes의 SIG Network와 보안 대응 위원회는 Ingress NGINX의 은퇴를 발표했습니다. Ingress NGINX는 2026년 3월까지 최선의 노력을 다해 유지 보수될 예정이며, 그 이후로는 더 이상의 릴리스나 버그 수정, 보안 취약점 해결 업데이트가 제공되지 않습니다. 기존의 Ingress NGINX 배포는 계속 작동하며 설치 아티팩트도 계속 이용 가능합니다. 사용자들은 Gateway API 또는 다른 Ingress 컨트롤러로의 마이그레이션을 권장받고 있습니다. Ingress NGINX는 Kubernetes 초기에 개발된 Ingress 컨트롤러로, 유연성과 기능의 폭이 넓어 많은 인기를 끌었으나, 유지 보수 인력 부족 등의 문제로 인해 은퇴가 결정되었습니다. 2026년 3월 이후로 GitHub 리포지토리는 읽기 전용으로 전환됩니다. 사용자들은 즉시 대체 솔루션으로의 전환을 시작하는 것이 권장됩니다.
👉 [자세히 보기](https://kubernetes.io/blog/2025/11/11/ingress-nginx-retirement/)

## 🔹 Spring Boot - Spring Modulith 2.0 GA, 1.4.5, and 1.3.11 released
Spring Modulith 2.0 GA 버전이 출시되었으며, 이 릴리스는 Spring Modulith의 첫 번째 세대에서 얻은 모든 학습을 통합한 중요한 이정표입니다. 주요 기능으로는 이벤트 발행 생명 주기 개편, 애플리케이션 모듈별 Flyway 마이그레이션 지원, 이벤트 외부화의 직렬 실행 허용, 이벤트 직렬화에서 Jackson 3 지원, 애플리케이션 모듈 구조를 시작 시 검증하는 기능 등이 포함되어 있습니다. 또한 Spring Boot 4와 Framework 7로의 업그레이드가 이루어졌습니다. 더 자세한 내용은 향후 블로그 게시물에서 확인할 수 있으며, 버그 수정 릴리스도 제공됩니다.
👉 [자세히 보기](https://spring.io/blog/2025/11/21/spring-modulith-2-0-ga-1-4-5-and-1-3-11-released)
