## 🔹 Kubernetes - Ingress NGINX Retirement: What You Need to Know
Kubernetes SIG Network와 Security Response Committee는 Ingress NGINX의 퇴역을 발표했습니다. 2026년 3월까지 유지 보수가 이루어지며, 이후로는 추가 릴리스나 버그 수정, 보안 취약점에 대한 업데이트가 제공되지 않습니다. 기존의 Ingress NGINX 배포는 계속 작동하며 설치 아티팩트도 계속 제공됩니다. 사용자는 Gateway API 또는 다른 Ingress 컨트롤러로의 마이그레이션을 권장받고 있습니다. Ingress NGINX 프로젝트는 유지 보수 인력 부족으로 인해 지속 가능하지 않으며, 2026년 3월에 공식적으로 퇴역할 예정입니다. SIG Network와 Security Response Committee는 모든 사용자가 즉시 Gateway API나 다른 Ingress 컨트롤러로 마이그레이션을 시작할 것을 권장합니다.
👉 [자세히 보기](https://kubernetes.io/blog/2025/11/11/ingress-nginx-retirement/)

## 🔹 Spring Boot - Spring Modulith 2.0 GA, 1.4.5, and 1.3.11 released
Spring Modulith 2.0이 출시되었습니다. 이 버전은 첫 번째 세대의 학습을 바탕으로 중요한 기능 개선을 포함하고 있습니다. 주요 기능으로는 이벤트 발행 라이프사이클의 전면 개편, 애플리케이션 모듈별 Flyway 마이그레이션 지원, 이벤트 외부화의 직렬 실행 허용, Jackson 3 지원, 애플리케이션 모듈 구조 검증 지원 등이 있습니다. 또한, Spring Boot 4와 Framework 7로의 기반 업그레이드가 이루어졌으며, 더 상세한 내용은 추후 블로그 포스트로 제공될 예정입니다. 버전 1.4 및 1.3의 버그 수정 릴리스도 함께 배포되었습니다. 자세한 사항은 변경 로그에서 확인할 수 있습니다.
👉 [자세히 보기](https://spring.io/blog/2025/11/21/spring-modulith-2-0-ga-1-4-5-and-1-3-11-released)
