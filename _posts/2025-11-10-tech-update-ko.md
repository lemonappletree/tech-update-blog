# 🛠️ 2025-11-10 기술 업데이트 요약

## 🔹 Kubernetes - Gateway API 1.4: New Features
이 블로그 글은 Kubernetes의 Gateway API v1.4.0 버전의 새로운 기능을 소개합니다. 2025년 10월 6일에 출시된 이 버전은 Kubernetes 네트워킹을 현대적이고 표현력이 풍부하며 확장 가능한 서비스 네트워킹으로 강화합니다. 주요 기능으로는 게이트웨이와 백엔드 간의 TLS 정책, GatewayClass 상태의 지원 기능 표시, 그리고 라우트에 대한 명명된 규칙을 포함합니다. 또한 서비스 메시 구성용 메쉬 리소스, 구성을 쉽게 해주는 기본 게이트웨이, 그리고 HTTPRoute에 대한 외부 인증 필터 등 새롭게 실험적인 기능들이 추가되었습니다. 이번 업데이트는 보안 강화 및 사용 편의성을 높이기 위한 여러 가지 개선 사항을 포함하고 있으며, 이를 통해 개발자와 운영자가 게이트웨이 API를 더 효과적으로 활용할 수 있도록 지원합니다.
👉 [자세히 보기](https://kubernetes.io/blog/2025/11/06/gateway-api-v1-4/)

## 🔹 Spring Boot - Spring AI 1.1.0-RC1 Available Now
Spring AI 1.1.0-RC1 버전이 출시되어 Maven Central에서 다운로드할 수 있습니다. 이번 패치 릴리스는 안정성 개선 및 버그 수정에 중점을 두었으며, 40개의 개선 사항과 버그 수정, 문서 업데이트를 포함합니다. 주요 개선 사항으로는 기능 확장, 커뮤니티에서 보고된 문제 해결, 보안 강화를 위한 의존성 업그레이드 등이 있습니다. 또한 Spring AI의 주요 기능 영역에서도 다양한 개선이 이루어졌습니다. 향후 Spring AI 팀은 Spring Boot를 통한 AI 애플리케이션 개발을 계속 발전시킬 계획이며, 1.1.0 GA 버전도 곧 출시될 예정입니다.
👉 [자세히 보기](https://spring.io/blog/2025/11/08/spring-ai-1-1-0-RC1-available-now)

## 🔹 Java - Pulling the (Foreign) String
기술 블로그 "Pulling the (Foreign) String"에서는 Foreign Function & Memory (FFM) API의 문자열 처리 기능을 다루고 있습니다. FFM API는 메모리 세그먼트에서 문자열을 읽고 쓰거나, 기존 Java 문자열로부터 메모리 세그먼트를 할당하는 메서드를 제공합니다. 이 글에서는 문자열과 메모리 세그먼트 간의 상호 운용성을 더 효율적으로 지원하기 위한 FFM API의 개선 방안을 탐구합니다.
👉 [자세히 보기](https://inside.java/2025/11/08/ffm-string/)
